ASIX測試程式版本及修正內容

ASIXTEST ver 1.0.9
建構日期:2017/06/02
修改項目:
1.修改fping的方式，改為ping，一樣三次為限，比較穩定。
2.修改WRITECLICK，將燒完MAC後重啟拿掉，避免掉燒入及測試失敗，也可能藍頻。

ASIXTEST ver 1.0.9
建構日期:2017/06/02
修改項目:
1.加入限制88772只能在Windows 7上測試，Test Driver切換會藍頻
2.AutoRun設定和補燒反灰
3.READClick 加入，驅動無法開啟時用GetMAC去撈MAC號
4.修改沒有Test Driver也可測試網路
5.修改fping 回應秒數200->1000 數量10->3 解決88772 ping太快掉封包問題
6.插入時會去偵測插入消息是否有符合的PVID方能起測，解決拔差隨身碟會起測問題
